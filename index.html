O que é esse site?

 Esse site tem como objetivo fornecer diversos métodos numéricos por meio de uma interface acessível. A ideia por trás desse projeto é construir, "from scratch", um servidor HTTP com a linguagem C para servir um site estático no "localhost" e, através de uma API, permitir que o JS atualize as páginas com WebSockets implementados em C.

 No final desse projeto, pretendo ter alcançado: um conhecimento além do básico sobre networking com C no Windows; criação de APIs que permitam que o frontend utilize os métodos numéricos servidos pelo servidor em C; habilidade de manipular HTML, CSS e JS para criar páginas completas; conhecimento razoável sobre cálculo numérico com C e criação de gráficos com Javascript.
